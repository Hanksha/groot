# GROOT

GROOT is a Rest API that approximates the root of a function using various algorithms.

## Features

### Algorithms
* Bisection
* Newton-Raphson
* Regula falsi
* Secant

### Operators
* \+
* \-
* \*
* /
* % (mod)
* ^ (power)

### Functions
* cos
* sin
* tan
* max
* min
* log
* abs
